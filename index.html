Filen körs lokalt (file://) vilket blockerar externa API-anrop. Två enkla lösningar:
Alternativ 1 — GitHub Pages (rekommenderas):
1. Skapa ett gratis konto på github.com
2. Skapa ett nytt repository → ladda upp denna HTML-fil
3. Gå till Settings → Pages → välj branch main → spara
4. Öppna din GitHub Pages-URL — API-hämtning fungerar direkt ✓

Alternativ 2 — Lokal Python-server (om Python är installerat):
Öppna Terminal i samma mapp som filen och kör:
python3 -m http.server 8080
Öppna sedan http://localhost:8080 i webbläsaren ✓

Tills dess: Använd manuell inmatning nedan — fungerar alltid utan server.
